Фронтенд-часть сайта для учебного центра "Learn & Grow"

#Структура проекта:

```
.
└── .../
    ├── .env
    ├── dist/ - Сборка проекта
    ├── public/ - Каталог, в котором лежат файлы, изменяемые и перемещаемые бандлером
    └── src/ - Корневая папка с кодом
        ├── assets/
        ├  ├── fonts/ - шрифты
        ├  ├── icons/ - иконки svg
        ├  └── images/ - картинки png 
        ├── enums/ - переиспользуемые enum
        ├── navigation/ - навигация
        ├── screens/ - страницы
        ├── services/ - сервисы (HTTP)
        ├── shared/ - Переиспользуемые компоненты, кастомные хуки
        ├── store/ - state
        ├── styles/ - глобальные стили
        ├── types/ - глобальные типы
        ├── utils/ - утилиты
        ├── App.tsx
        ├── config.ts - конфиг
        ├── vite-env.d.ts
        └── main.tsx

```
